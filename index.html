🖼️ imageUrl — works with ANY question type

Add a public image URL to the imageUrl column to show an image above that question's input area. Works with radio, checkbox, text, etc. Use Google Drive, Imgur, or any direct image link ending in .jpg/.png/.webp.

📍 hotspot — data format

User clicks a defined zone on an image. Zones are defined as percentages so they work on any screen size.
imageUrl: the image to click on (required for hotspot)
options: pipe-separated zones — each zone: Label:x%,y%,w%,h%
correct: the Label of the correct zone

imageUrloptionscorrect
https://…/world-map.jpgEurope:30%,20%,18%,20%|Asia:55%,18%,28%,35%|Americas:10%,15%,22%,50%Europe

Zone format: Label:left%,top%,width%,height% — percentages relative to image container.

✅ mtf (Multiple True/False) — data format

A list of statements — user marks each one True or False independently.
options: pipe-separated statements
correct: pipe-separated TRUE/FALSE answers, one per statement (same order)

questiontypeoptionscorrect
Are these statements true or false?mtfThe sun is a star|Water boils at 50°C|DNA has two strandsTRUE|FALSE|TRUE

🔲 matrix (Grid / Likert scale) — data format

A grid where each row is a statement/item and each column is a rating/option. User picks one cell per row.
options: pipe-separated row labels
matrixCols: pipe-separated column headers (new column)
correct: optional — pipe-separated Row::Column pairs for scoring

questiontypeoptions (rows)matrixColscorrect (optional)
