🚀 Hướng dẫn cài đặt

Hoàn chỉnh trong 4 bước

Không cần hosting, không cần server, không cần biết code.

1

Tạo Google Sheet

Tạo tab Tests và tab câu hỏi. Điền theo mẫu có sẵn.

2

Dán Apps Script

Mở Extensions → Apps Script, dán đoạn code Code.gs và lưu.

3

Deploy Web App

Deploy → New deployment → Web App. Chọn Anyone. Sao chép URL /exec.

4

Mở & dùng ngay

Mở index.html trên trình duyệt — bài kiểm tra tự tải về. Chia sẻ cho học sinh.
